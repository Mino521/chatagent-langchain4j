# Chat Agent based on Langchain4j

This is a chat agent based on Langchain4j. Default LLM is GPT-4o-mini.<br>
Support Ollama for better data security and scalability.

Main tech stack:<br> 
- Java + Springboot + Langchain4j<br>
- MySQL + Hibernate<br>
- MongoDB
